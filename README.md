# 📁 File Manager & Search Tool

## 📌 Overview

This project is a File Manager and Search Tool built using Java and JavaFX. It allows users to efficiently search for files and folders within a system and display their full paths through a simple and minimal user interface.

The main goal of this project is to provide fast file searching with a clean UI while focusing more on functionality and backend logic.

---

## 🚀 Features

🔍 Search files by name  
📂 Display full file paths  
⚡ Fast search using optimized logic  
🖥️ Simple JavaFX UI (Search bar + results list)  
🔗 Connected frontend and backend search engine  

---

## 🛠️ Tech Stack

Java – Core logic  
JavaFX – User Interface  
VS Code – Development environment  

---

## ⚙️ How It Works

- User enters a file name in the search bar  
- Backend searches through system directories  
- Matching results are collected  
- Navigation is used through folders  
- Results are displayed in JavaFX UI (ListView)  

---

## ▶️ Setup & Run (VS Code)

### Install Requirements
- Java JDK (17+ recommended)  
- JavaFX SDK  
- VS Code Extensions: Java Extension Pack  

---

## ⚙️ Configure JavaFX

Add VM options:

--module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml

## ⚠️ Important Note (VERY IMPORTANT)

This project contains hardcoded file/folder paths everywhere, so please update them according to your system before running. Placeholder support was planned… but let’s just say motivation didn’t compile
